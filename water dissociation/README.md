🧪 Исследовательский агент: Анализ научных статей (ArXiv + Llama 3.3)
Этот проект представляет собой автоматизированный воркфлоу в n8n, который самостоятельно ищет научные статьи на ArXiv, переводит их заголовки и генерирует краткую выжимку (саммари) с помощью ИИ. Результат отправляется в Telegram-бот в виде красиво оформленного отчета.

🚀 Основные функции
Автоматический поиск: Сбор свежих публикаций по ключевым словам (например, «dissociation» или «water») напрямую из репозитория ArXiv.

Локализация: Перевод заголовка статьи на русский язык для удобства восприятия.

Интеллектуальный анализ: Использование модели Llama-3.3-70b через API Groq для создания глубокой аннотации и выделения сути исследования.

Telegram-уведомления: Доставка готового отчета с использованием HTML-разметки для стабильной работы ссылок и спецсимволов.

🛠 Технологический стек
n8n: Платформа для автоматизации и связки узлов.

Groq Cloud (Llama 3.3): Высокоскоростной ИИ для обработки текста.

ArXiv API: Источник актуальных научных данных.

Telegram Bot API: Канал доставки контента.

JavaScript: Кастомная очистка и трансформация данных внутри узлов.

📦 Структура воркфлоу
Trigger: Запуск по расписанию или вручную.

ArXiv Search: HTTP-запрос к API для получения XML-фида со статьями.

Translation: Перевод названия статьи.

Data Cleaning: JavaScript-узел для извлечения ID, ссылок и аннотаций из сырого XML.

AI Analysis: Отправка данных в Groq для генерации выжимки.

Telegram Output: Форматирование финального сообщения в HTML-режиме.

🔧 Настройка
Импортируйте JSON воркфлоу в n8n.

Создайте Credentials для Telegram (через @BotFather) и Groq Cloud.
