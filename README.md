🌦 AI-Weather Agent (n8n + Telegram)
Проект по автоматизации сбора данных о погоде и отправке персонализированных уведомлений в Telegram.

📋 Описание
Агент автоматически запускается каждый день в 8:00 утра, запрашивает актуальный почасовой прогноз погоды через API Open-Meteo и отправляет структурированный отчет, разделенный на три периода: Утро, День и Вечер.

🛠 Технологический стек
n8n — платформа для автоматизации рабочих процессов (Workflows).

JavaScript (Node.js) — обработка и форматирование данных в узлах Code.

Open-Meteo API — источник метеорологических данных (почасовой прогноз).

Telegram Bot API — интерфейс взаимодействия с пользователем.

⚙️ Функционал
Автоматический триггер: Запуск по расписанию (Schedule Trigger).

Сегментация данных: Фильтрация почасового прогноза на логические блоки (Утро: 6-11, День: 12-17, Вечер: 18-23).

Динамический расчет: Поиск минимальной и максимальной температуры внутри каждого временного интервала.

Визуализация: Использование эмодзи и HTML-разметки для удобного чтения в мессенджере.

📦 Установка и запуск
Импортируйте файл weather_bot_n8n.json в свой экземпляр n8n.

В узле "Запрос к api погоды" укажите ваши географические координаты (lat/lon).

Настройте подключение (Credentials) для вашего Telegram-бота.

Укажите ваш Chat ID в параметрах узла отправки сообщения.
